# 现代化管理端页面开发计划

## 项目背景
- 项目：ChinaJoin CMS 内容管理系统
- 技术栈：Nuxt 3 + Vue 3 + TypeScript
- 开发流程：开发 → 测试 → 修复 → 提交推送
- UI框架：Nuxt UI (Naive UI 风格)
- 富文本编辑：Tiptap + 实时预览

## 开发顺序
1. 基础设施搭建 → 2. 新闻管理 → 3. 媒体管理 → 4. 用户管理 → 5. 网站设置 → 6. 实时预览
